Existem vários "Tipos" de IA!

Assim como existem animais diferentes (um peixe 🐠 e um macaco 🐒), existem IAs diferentes.

Vamos ver 3 NÍVEIS de IA:

NÍVEL 1: A IA "ESPECIALISTA" 🤓

(É a IA que temos HOJE!)

Essa IA é super boa em fazer UMA coisa. Mas ela só sabe fazer isso!

Ex: O GPS (só sabe mapas 📍), a Netflix (só sabe de filmes ▶️).

NÍVEL 2: A IA "DOS FILMES" 🤖

(Ainda NÃO EXISTE de verdade!)

Seria uma IA inteligente como um humano. Que sabe conversar, aprender, jogar E fazer matemática.

Os cientistas estão trabalhando muito para tentar criar uma!

(É só uma IDEIA para o futuro distante!)

Seria uma IA mais inteligente que TODOS os humanos do planeta juntos!

Como falar com a IA?

Nós não "programamos" o Gemini ou o ChatGPT. Nós "pedimos".

Esse pedido tem um nome especial: O "Prompt"! 🪄

O que é um "PROMPT"? 📝

É a sua INSTRUÇÃO. O seu PEDIDO. A sua ORDEM.

O Prompt é como você diz à IA o que ela deve fazer.

"Prompt" = "As Palavras Mágicas"

O Pedido Mágico 🪄

❌ Prompt RUIM

"Faz um desenho."

🤔

(A IA fica confusa: "Desenho de quê?")

✅ Prompt BOM

"Desenhe um gato laranja, gordo, com um chapéu de pirata, num navio."

🐱‍☠️

(A IA entende exatamente!)

A Fórmula do "Bom Prompt"

SEJA O CHEFE! 👑 Dê DETALHES!

  • O Quê? (A tarefa: "Escreva", "Crie", "Resuma")
  • Quem? (O Estilo: "Como um pirata", "Para uma criança")
  • Detalhes! (O Tema: "Sobre dinossauros", "Em 3 partes")

Exercício: O Prompt! ✍️

Que "Prompt" (pedido) você daria a uma IA (como eu) para criar este jogo?

Vamos ver dois exemplos...

❌ Prompt RUIM

"Faz um jogo para mim."

🤯

(A IA não sabe que jogo, que regras, que cor...)

✅ Prompt BOM (O Pedido Mágico!)

"Crie um jogo completo em HTML, CSS e JavaScript num só ficheiro.
1. O jogador é um robô 🤖.
2. Ele voa para cima quando o jogador clica.
3. Canos verdes 🔧 aparecem da direita.
4. Se o robô bater num cano ou no chão, o jogo recomeça.
5. Mostre a PONTUAÇÃO (quantos canos passou)."

E a IA... FAZ! 🪄

Com o "Prompt Bom", a IA entende e escreve todo o CÓDIGO do jogo!
